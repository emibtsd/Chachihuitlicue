# Práctica 11 - Actividad 2

Este apartado contiene el desarrollo de la segunda actividad correspondiente a la Práctica 11.

## Contenido de la carpeta
* **`README.md`**: Descripción de la actividad.
* **`Practica11.2.pdsprj`**: Simulación del circuito en Proteus.
* **`Practica_11_2.X.production.hex`**: Archivo compilado con el código del ejercicio.
* **`Actividad 11.2.png`**: Captura de pantalla del circuito funcionando en el simulador.

## Descripción del Ejercicio
Consiste en controlar de forma independiente la intensidad luminosa de dos LEDs mediante la lectura de dos potenciómetros. El primer LED se regula utilizando un módulo PWM por hardware propio del microcontrolador, mientras que el segundo LED se controla implementando una señal PWM generada completamente por software a través del código.
