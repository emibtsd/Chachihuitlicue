# Práctica 10 - Clase

Este apartado contiene los archivos desarrollados durante la sesión de clase de la Práctica 10.

## Contenido de la carpeta
* **`README.md`**: Descripción de la actividad.
* **`Clase 10.png`**: Captura de pantalla del circuito funcionando en el simulador.

## Descripción del Ejercicio
Consiste en la configuración y uso del temporizador interno Timer 1 para llevar un registro preciso del tiempo transcurrido desde el inicio del programa y desplegar de forma dinámica este tiempo de ejecución del microcontrolador en la pantalla LCD 16x2.
