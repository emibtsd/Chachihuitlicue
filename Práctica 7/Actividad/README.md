# Práctica 7 - Actividad

Este apartado contiene el desarrollo de la actividad correspondiente a la Práctica 7.

## Contenido de la carpeta
* **`README.md`**: Descripción de la actividad.
* **`Actividad 7.png`**: Captura de pantalla del circuito funcionando en el simulador.

## Descripción del Ejercicio
Consiste en un sistema de monitoreo analógico que lee la señal de un potenciómetro y permite al usuario, mediante la presión de un botón físico, intercalar de forma cíclica la visualización en la pantalla LCD entre tres formatos diferentes: el valor bruto del ADC, el voltaje calculado (V) y el porcentaje equivalente (0% a 100%).
