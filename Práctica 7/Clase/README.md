## Descripción del Ejercicio
Consiste en realizar la lectura analógica de un potenciómetro utilizando el convertidor analógico-digital (ADC) del microcontrolador y desplegar de forma continua el valor del voltaje correspondiente en la pantalla LCD 16x2.
