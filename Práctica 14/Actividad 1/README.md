# Práctica 14 - Actividad 1

Este apartado contiene el desarrollo de la primera actividad correspondiente a la Práctica 14.

## Contenido de la carpeta
* **`README.md`**: Descripción de la actividad.
* **`Practica14.pdsprj`**: Simulación del circuito en Proteus.
* **`Practica_14.X.production.hex`**: Archivo compilado con el código del ejercicio.
* **`Actividad 14.1.png`**: Captura de pantalla del circuito funcionando en el simulador.

## Descripción del Ejercicio
Consiste en un sistema de control automático que varía de forma continua el ciclo de trabajo de una señal PWM para hacer que un servomotor realice un giro secuencial y fluido de 0° a 180° y, posteriormente, regrese a su posición inicial.
