# Práctica 14 - Actividad 2

Este apartado contiene el desarrollo de la segunda actividad correspondiente a la Práctica 14.

## Contenido de la carpeta
* **`README.md`**: Descripción de la actividad.
* **`Practica14.2.pdsprj`**: Simulación del circuito en Proteus.
* **`Practica_14_2.X.production.hex`**: Archivo compilado con el código del ejercicio.
* **`Actividad 14.2.png`**: Captura de pantalla del circuito funcionando en el simulador.

## Descripción del Ejercicio
Consiste en un sistema de posicionamiento manual que lee el voltaje analógico de un potenciómetro y mapea dicho valor en el código para controlar el ángulo exacto de giro de un servomotor en tiempo real de acuerdo a la posición del potenciómetro.
