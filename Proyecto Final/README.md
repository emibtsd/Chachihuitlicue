# Proyecto Final

Este apartado contiene todos los archivos del sistema embebido integrado desarrollado como proyecto final del curso.

## Contenido de la carpeta
* **`README.md`**: Descripción de la actividad.
* **`finalmicro.pdsprj`**: Simulación completa del proyecto en Proteus.
* **`final.X.production.hex`**: Archivo compilado con el firmware del sistema.
* **`Final Micro.jpeg`**: Captura de pantalla del circuito funcionando en el simulador.

## Descripción del Ejercicio
Consiste en el diseño e implementación de un videojuego interactivo que utiliza una matriz de LEDs de 8x8 como pantalla de juego principal, controlada en tiempo real mediante dos mandos de tipo Joystick. El sistema cuenta además con una pantalla LCD 16x2 para mostrar estadísticas o estados, y una bocina/buzzer integrada para emitir una alerta sonora en el momento en que el usuario pierde la partida.
