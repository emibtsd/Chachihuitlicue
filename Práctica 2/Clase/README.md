# Práctica 2 - Clase

Este apartado contiene los archivos desarrollados durante la sesión de clase de la Práctica 2.

## Contenido de la carpeta
* **`README.md`**: Descripción de la actividad.
* **`Practica 2_actividad`**: Captura de pantalla del circuito funcionando en el simulador.
* **`Practica 2.pdsprj`**: Archivo de la simulacion en proteus.

## Descripción del Ejercicio
Consiste en el control y encendido coordinado de una matriz de LEDs de 8x8 para mapear sus filas y columnas con el fin de desplegar de forma estática la figura de una letra "X".
