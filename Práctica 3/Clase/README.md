# Práctica 3 - Clase

Este apartado contiene los archivos desarrollados durante la sesión de clase de la Práctica 3.

## Contenido de la carpeta
* **`README.md`**: Descripción de la actividad.
* **`Practica 9_clase.png`**: Captura de pantalla del circuito funcionando en el simulador.
* **`Practica 9_clase.pdsprj`**: Simulación del circuito realizado con proteus.
* **`Practica_9.X.production.hex`**: Archivo con la informacion en hexadecimal cargado al PIC.

## Descripción del Ejercicio
Consiste en el diseño y programación de un contador numérico automático que avanza de forma secuencial en base decimal desde el número 0 hasta el 9, visualizado a través de un display de 7 segmentos.
