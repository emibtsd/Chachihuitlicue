# Práctica 3 - Actividad

Este apartado contiene el desarrollo de la actividad correspondiente a la Práctica 3.

## Contenido de la carpeta
* **`README.md`**: Descripción de la actividad.
* **`Practica 3_actividad.png`**: Captura de pantalla del circuito funcionando en el simulador.
* **`Practica 3_actividad.pdsprj`**: Simulación del circuito realizado con proteus.
* **`Practica_3.X.production.hex`**: Archivo con la informacion en hexadecimal cargado al PIC.

## Descripción del Ejercicio
Consiste en un sistema de conteo que abarca toda la escala hexadecimal, mostrando de manera cíclica y secuencial los caracteres desde el 0 hasta la letra F (0, 1, 2... 9, A, B, C, D, E, F) utilizando un display de 7 segmentos.
