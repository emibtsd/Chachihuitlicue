## Descripción del Ejercicio
Consiste en la implementación de una calculadora básica interactiva. El sistema procesa la entrada de datos numéricos y la selección de operaciones a través de un teclado matricial, realiza los cálculos aritméticos correspondientes (suma, resta, multiplicación y división) y muestra el procedimiento y resultado final en una pantalla LCD 16x2.
