# Práctica 12 - Actividad 1

Este apartado contiene el desarrollo de la primera actividad correspondiente a la Práctica 12.

## Contenido de la carpeta
* **`README.md`**: Descripción de la actividad.
* **`Actividad 12.1.png`**: Captura de pantalla del circuito funcionando en el simulador.
* *(Nota: Si tienes aquí tus archivos .pdsprj o .hex, puedes agregarlos a esta lista)*

## Descripción del Ejercicio
Consiste en un sistema de interfaz de usuario que realiza el escaneo de filas y columnas de un teclado matricial para detectar la pulsación de cualquier tecla y desplegar instantáneamente el carácter correspondiente en una pantalla LCD 16x2.
