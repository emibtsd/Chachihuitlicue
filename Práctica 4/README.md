# Práctica 4: Entradas Digitales

## 📝 Clase
Usar 3 botones conectados al puerto B para encender 3 LEDs conectados en otro puerto. Utilizar las resistencias de Pull-up internas.
* [Clase](./Clase)

## ⚡ Actividad 
Realizar un contador de 0 a 99 con dos displays de 7 segmentos. Utilizar un boton que sume una unidad y otros que reste una unidad. Utilizar un boton de reset.
